# SoulCare — Mental Wellness Prototype

A clean, mobile-first, polished prototype of a mental health web app.
Built with plain HTML / CSS / JavaScript. Uses `localStorage` — no backend.

## Features
- Mood tracker (emoji quick-save)
- Journal (save personal notes)
- Meditation (breathing circle + audio)
- Profile with export & clear data
- Responsive, mobile-first design with animations & icons

## How to run locally
1. Open the project folder in VS Code.
2. Install the Live Server extension and open `index.html` with Live Server.
3. Or open `index.html` directly in a browser.

## Deploy
- Create a GitHub repo and push the project.
- Enable GitHub Pages from `main` branch → `/ (root)` OR use Vercel/Netlify (drag & drop).

## Notes
- Data is stored locally in the browser (localStorage).
- Export available as JSON for backup.
